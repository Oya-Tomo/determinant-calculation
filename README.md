## description

`calc.py` contains the function that can calculate NxN matrix.

```python
ary = [
    [3, 1, 2, 4, 5],
    [3, 0, 3, 2, 3],
    [1, 1, 5, 2, 3],
    [1, 2, 0, 5, 2],
    [4, 1, 3, 2, 1],
]
get_det(ary)
# output => 155
```
